# Discount Pharmacy – Medical Shop Stock Management System

## Overview
Discount Pharmacy is a Medical Shop Stock Management System developed to manage medicine inventory, billing, expiry tracking, and sales reporting efficiently.  
This project demonstrates practical application of frontend design and backend logic integration.

## Functional Modules
- Login Module
- Add Medicine Stock
- Update Existing Stock
- View Available Stock
- Billing System
- Expiry Date Check
- Sales Report Generation

## Technologies Used
- **HTML** – User interface
- **CSS** – Styling and layout
- **C Programming** – Core logic and data handling

## How to Run
1. Download or clone the repository.
2. Open **`login.html`** in a web browser.
3. Use the login interface to access the system.
4. Navigate through different modules using the UI.

## Data Handling
- Stock and billing data are handled using appropriate data structures in C.
- The system demonstrates structured programming and modular design.
